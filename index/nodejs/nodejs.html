# 将自己的nodeJS项目分享到npm上

# 一、在npm官网上注册一个账号：

npm官网链接

提示

注意注册完账号记得绑定邮箱，如果没有绑定邮箱的话无法上传npm包。

# 二、生成package.json文件

  1. 在项目根目录文件下打开控制台，使用npm init生成package.json文件。下面是打开控制台的几种方法。
    第一种：在项目中按住shift+右键，点击在此处打开命令窗口。
    第二种：在项目中输入cmd敲回车
    第三种：使用按住电脑上的window+r，进入如下图所示，然后输入cmd，点击确定进入控制台
    第四种：在开始菜单里输入cmd，运行控制台，不过进入控制台后，如果需要到你的项目中，则需要使用命令cd “项目所在文件夹路径”注意命令cd和项目所在文件夹路径中间有个空格进入项目文件夹下
    下图为上传生成pack.json文件的一个示列：

# 三、在项目中添加用户

package.json文件生成好了以后，使用命令npm adduser在项目中添加用户信息

# 四、将项目发布至npm上

  1. 在给项目添加完用户信息以后，使用命令npm publish将自己的项目发布至npm上。
  2. 发布成功以后就可以在npm官网上查到你的包了。

# 五、如何删除发布至npm上的包

在项目中使用命令npm unpublish --force即可
